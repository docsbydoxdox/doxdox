## Install

```bash
$ npm install doxdox-cli@v4.0.0-preview.24 doxdox-renderer-template@v4.0.0-preview.24 --save-dev
```

## Usage

```bash
$ doxdox -r template > docs.txt
```

or

```bash
$ doxdox -r template -o docs.txt
```
